## 📌 一句话总结

> 针对动物叫声增强，提出多尺度双轴注意力与谐波增强模块，在三个数据集上以更低计算量达到或超越SOTA语音增强模型。

## 📖 阅读建议

适合做语音增强或生物声学的研究者阅读。建议重点看§3的模块设计（多尺度双轴注意力、谐波增强、能量自适应门控）和§4的实验结果。如果对动物声音增强感兴趣，值得通读全文。

## 🧩 模型一栏概览

| 项 | 内容 |
| --- | --- |
| 应用场景 | 野外动物声音记录增强，用于生物多样性监测与保护。 |
| 核心创新 | 多尺度双轴注意力单元提取时频特征 · 生物谐波多尺度增强单元捕获谐波结构 · 能量自适应门控连接单元防止误删目标叫声 |
| 模型架构 | 输入时频谱 → 多尺度双轴注意力单元（沿时间和频率轴分别注意力）→ 生物谐波多尺度增强单元（多尺度卷积+谐波建模）→ 能量自适应门控连接单元（频率权重门控）→ 输出增强时频谱。参数量未给出。 |
| 数据集 | Bioacoustic dataset A（训练/评估） · Bioacoustic dataset B（训练/评估） · Bioacoustic dataset C（训练/评估） |
| 关键结果 | 在三个生物声学数据集上，BioSEN的PESQ/STOI等指标匹配或超过SOTA语音增强模型（如SEPFormer等），但计算量显著更低。摘要未给出具体数值。 |

## 🎯 与本站重点领域的关联

属于语音增强领域，但面向生物声学而非人类语音。其多尺度双轴注意力与谐波增强模块可迁移至人类语音增强任务，尤其对处理音乐或语音中的谐波结构有借鉴意义。
